HTML Image Elements


· HTML의 <img> 태그 기능을 통해 이미지를 표시할 수 있다.

· 이미지를 단독으로 표시하거나 이미지를 통해 하이퍼링크를 연결하는 등으로 활용할 수 있다.


1. <img src="house.jpg" width="300" height="230"> - 이미지의 소스는 src 속성을 통해 지정한다. - 이미지의 크기는 width, height 속성을 통해 지정한다. 2. <img src="house.jpg" alt="A Nice House"> - src에 지정된 이미지 표시에 실패했을 때, alt를 통해 대체 텍스트나 이미지를 표시할 수 있다. 3. <a href="photo1.html"> <img src="1.jpg" alt=""> </a> - <a href> 태그 사이에 <img> 태그를 사용하여 이미지에 하이퍼 링크를 연결하여, 지정된 이미지를 클릭하면 다른 페이지로 이동할 수 있다.
